# SortMaster
Library for .NET applications with multiple sorting algorithms and utils

- Algorithms
   - BubbleSort
   - CocktailSort
   - CombSort
   - CycleSort
   - GnomeSort
   - HeapSort
   - InsertionSort
   - MergeSort
   - QuickSort
   - SelectionSort
   - ShellSort
   - TimSort
  (ONLY FOR UINT)
   - CountSort
   - PigeonholeSort
   - RadixSort
   
- Utils
   - SortingManager (Contains all sorting algorithms in one class)
